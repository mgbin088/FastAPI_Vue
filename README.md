

该项目是一个个人学习用的模板项目，用来找工作的

作者是一名面向百度，github的菜鸟程序猿. 请不要对该项目抱有太高期望




项目框架：
  vue3+fastapi(python3.7.15+)+mongodb  


前端启动：

    cd client
    npm install
    npm run serve
    
    
后端启动：
    
    cd serverAPI
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python manage.py
